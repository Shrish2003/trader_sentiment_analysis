## Trader Performance vs Market Sentiment

### Objective
Analyze how market sentiment (Fear, Greed, Neutral) impacts trader behavior
and performance on Hyperliquid.

### Methodology
- Cleaned and aligned sentiment and trade data at daily level
- Normalized sentiment regimes
- Computed PnL, win rate, and trade frequency
- Analyzed performance and behavior across sentiment regimes

### Key Insights
1. Fear regimes show highest PnL volatility and downside risk
2. Greed regimes show increased activity with upside potential
3. Neutral regimes are most stable

### Strategy Recommendations
- Reduce leverage during Fear regimes
- Allow aggressive trading during Greed only for high win-rate traders

### How to Run
Open analysis.ipynb and run all cells.
